手机竞品分析 - 固定安装目录与桌面图标
========================================

方式一：仅创建桌面快捷方式（当前目录即“固定目录”）
-------------------------------------------------
1. 将本整个项目文件夹放到你希望固定的位置，例如：
   D:\PhoneCompetitor  或  C:\Program Files\手机竞品分析
2. 双击运行「创建桌面快捷方式.bat」
3. 桌面会出现「手机竞品分析」图标，双击即可启动

方式二：安装到固定目录并自动创建桌面图标
-------------------------------------------------
1. 双击运行「安装到固定目录.bat」
2. 脚本会：
   - 将项目复制到 C:\PhoneCompetitor（可编辑该 bat 修改为 D:\PhoneCompetitor 等）
   - 自动创建虚拟环境并安装 Python 依赖
   - 安装前端 npm 依赖
   - 在桌面创建「手机竞品分析」快捷方式
3. 以后双击桌面「手机竞品分析」即可从固定目录启动

修改固定安装路径：
打开「安装到固定目录.bat」，找到 set "TARGET=C:\PhoneCompetitor"，改为你想要的路径后保存。

需求：本机已安装 Python 3.10+ 和 Node.js，并已加入系统 PATH。
